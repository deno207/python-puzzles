# Puzzle

## Summary

Define a function `sum_if_less_than_fifty` that takes two integers and returns the sum if it is less than fifty, or
`None` if the sum is fifty or greater.

## Inputs

| Name    | Type | Example |
|---------|-----|---------|
| num_one | `int` | 20      |
| num_two | `int` | 25 |

## Outputs

| Type         | Example |
|--------------|---------|
| `int` or `None` | 45      |
